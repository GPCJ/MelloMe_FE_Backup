description: force push 등 외부 레포에 push하기 전 반드시 사용자 확인을 받을 것
type: feedback
---

외부 레포(GitHub 등)에 push — 특히 force push — 하기 전에 반드시 사용자 최종 승인을 받을 것. 자동으로 진행하지 말 것.

**Why:** airo remote에 force push가 필요한 상황에서 사용자가 "push는 아직 하지 말라, 최종 승인을 따로 해줄게"라고 명시적으로 요청함.

**How to apply:** push 준비가 완료되면 "push할 준비됐어요, 진행할까요?" 로 확인 후 진행. 이미 CLAUDE.md의 일반 원칙과도 일치하지만 이 프로젝트에서 특히 강조됨.
