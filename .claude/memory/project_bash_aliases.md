`~/.bash_aliases`에 Claude Code 바로가기 등록됨.

| 키워드 | 경로 |
|--------|------|
| `mel` | `/home/jin24/my-project` (main) |
| `mel-dev` | `/home/jin24/my-project/.claude/worktrees/feature-codegen` |
| `mel-review` | `/home/jin24/my-project/.claude/worktrees/code-review` |
| `portfolio` | `/home/jin24/portfolio` (포트폴리오 사이트) |

**Why:** 터미널에서 프로젝트별로 빠르게 Claude Code를 열기 위해 생성.
**How to apply:** 키워드 변경/추가 요청 시 `~/.bash_aliases` 파일 수정.
