웹 보안/인증 관련 개념(CORS, Origin, withCredentials, JWT 토큰 흐름)에 익숙하지 않음.
"origin이 뭔지는 모르겠지만"이라고 직접 언급함 (2026-03-22).

비유적 설명에 잘 반응함 — "신분증(Refresh Token) / 출입 티켓(Access Token)" 비유를 대화 중 스스로 도출.

**How to apply:** 인증/보안 개념 설명 시 기술 용어 나열보다 비유나 흐름 중심으로 설명할 것.
