## 프로젝트 목록

| 키 | 이름 | 용도 |
|----|------|------|
| `MEL` | melonnebuilders | 멜로미 서비스 작업 |
| `BUR` | burst! | 별개 서비스 (멜로미 아님) |
| `SCRUM` | My Software Team | 기타 |

멜로미 관련 이슈는 반드시 **MEL 프로젝트**에 생성.

## MEL 컨벤션

- **에픽 제목**: prefix 없이 한국어 제목만 (BUR은 `[EP숫자]` prefix 사용)
- **설명 구조**: `현황:` / `할 일:` / `완료 조건:`
- **라벨**: 미사용
- **우선순위**: Medium 기본
