이슈는 airo 레포(AIRO-offical/therapist_community_FE)에만 작성한다. 멜로미 레포에는 더 이상 이슈를 작성하지 않는다.

**Why:** 백엔드와의 소통 채널을 airo 이슈로 단일화하기로 결정 (04-10)
**How to apply:** 백엔드 요청, 버그 리포트, 기능 요청 등 모든 이슈는 airo 레포에만 작성. 기존 `feedback_github_issues_dual_repo.md`(두 레포 동시 생성) 정책 대체.
