description: 노션에 새 항목 작성 전 실제 페이지 번호를 반드시 확인해야 함
type: feedback
---

노션에 새 항목(트러블슈팅 번호 등)을 추가하기 전, 메모리에 기록된 번호를 그대로 쓰지 말고 반드시 `notion-fetch`로 실제 페이지를 확인한 후 번호를 결정할 것.

**Why:** 메모리는 #004까지라고 기록돼 있었지만 실제로는 #006까지 있었음. #005로 작성하려다 사용자가 직접 잡아줘야 했음. 메모리는 언제나 stale할 수 있음.

**How to apply:** 노션 페이지에 순번이 있는 항목을 추가할 때는 항상 `notion-fetch`로 현재 상태를 먼저 확인하고 다음 번호를 결정.
