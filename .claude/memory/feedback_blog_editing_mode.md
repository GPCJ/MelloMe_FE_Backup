글쓰기 중 "편집 모드 켠 채 쓰기" 루프에 빠지는 성향 → 초안과 편집을 물리적으로 분리한다.

**Why:**
- 남 시선("이게 이상하지 않나?")을 의식해 썼다 지웠다 반복하면 한 섹션에서 30분씩 갇힘
- 완성 임계치가 계속 올라가서 발행 자체가 지연/포기됨
- 첫 글일수록 이 루프가 세게 걸리고, 여기서 뽀개져야 주 1회 전략이 돌아감 (2026-04-22 1번째 글 작성 중 자각)

**How to apply:**
1. **초안 단계에선 되돌아가지 않는다** — 섹션 단위로 끝까지 쭉 씀. 맞춤법·어색한 문장·논리 점프 전부 그대로 두고 진행.
2. **편집은 초안 전체 완성 후 한 번만** — 이때 "남들이 볼 때" 체크리스트 돌림. 쓰기 단계에선 이 체크리스트를 켜지 않음.
3. **막히면 섹션 스킵** — 썼다 지웠다 루프가 시작되면 그 섹션 접고 다음 섹션으로. 순서 안 맞아도 OK (쓰기 쉬운 것부터).
4. **취업용이라서 더 주관적으로** — "객관적으로 맞춰야"가 아니라 선명한 본인 관점이 면접 방어력·기억잔존력 모두 높음. 평균치로 다듬은 글은 지원서 100개 중 하나가 됨.
