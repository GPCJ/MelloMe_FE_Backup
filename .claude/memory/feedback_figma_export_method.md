type: feedback
---

피그마 화면을 Claude에게 공유할 때 프레임 다중 선택 → Export (PNG, 2x)로 일괄 내보내기가 효율적.

**Why:** 피그마 전체 개요 스크린샷은 해상도가 낮아 텍스트/아이콘 식별 불가. Claude는 이미지 확대 기능 없음.

**How to apply:** 피그마 공유 시 개별 프레임을 PNG 2x로 export하여 zip으로 전달. 프레임 이름을 정리해두면 순서대로 파악 가능.
