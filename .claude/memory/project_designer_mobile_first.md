디자이너가 현재 모바일 해상도 기준으로 쭉 작업 중. 데스크탑 디자인은 별도로 나올 예정.

**Why:** 데스크탑 디자인 없이 PC용 UI를 임의 구현하면 재작업 발생 가능.

**How to apply:** 프론트는 모바일 기준으로 먼저 구현하고, 데스크탑은 디자인 확정 후 맞추기. 데스크탑 전용 UI 질문이 나오면 디자이너 확인 필요함을 안내.
