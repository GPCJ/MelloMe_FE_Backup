사용자가 새 세션 시작 시 figma.com 링크만 단독으로 보내면 MVP 요구사항 피그마 링크임.

**Why:** 매번 "이게 뭐야", "요구사항 저장해줘" 설명 없이 링크만 줄 테니 바로 인지하고 처리해달라고 요청함

**How to apply:**
1. 새 세션에서 figma.com 링크가 오면 → 요구사항 피그마로 인지
2. 바로 Figma 플러그인으로 접근해서 MVP 요구사항 읽기
3. 읽은 내용을 메모리에 저장 (wiki `mvp-req-001-019`에 업데이트)
4. MEMORY.md 업데이트
