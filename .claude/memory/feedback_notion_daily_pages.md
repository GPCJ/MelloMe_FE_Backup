---
name: Notion 날짜별 페이지 방식
description: 매일 작성하는 노션 페이지(TIL 등)는 날짜별 서브페이지로 생성해야 함
type: feedback
---

매일 작성하는 노션 페이지(TIL, 진행 상황 등 일자별 기록)는 날짜별로 서브페이지를 생성해서 구분한다.

**Why:** 토글 방식보다 페이지 방식이 내용이 많아질수록 깔끔하고, 페이지 단위 공유/링크가 가능하다는 이유로 사용자가 선택.

**How to apply:** `/update-til`, `/update-builders` 등 날짜별 기록 작성 시 메인 페이지에 직접 내용을 추가하지 말고, 해당 날짜 제목의 서브페이지를 생성해서 내용을 작성할 것.
