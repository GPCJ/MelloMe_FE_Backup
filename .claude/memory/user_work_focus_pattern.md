사용자는 여러 todo 중 하나를 선택해 그 작업만 집중해서 완성도 있게 끝내고 다음 단계로 넘어가는 스타일을 선호한다. 본인도 이 방식이 전체 맥락 이해에 유리하다고 인지.

**본인이 인지하는 패턴 (ADHD-like 자가 표현, 2026-04-22)**
- 작업 중 다른 작업·아이디어가 자꾸 머리에 스침
- 이를 놓칠까 봐 다른 작업으로 스위칭했다가 돌아오는 것이 반복
- 결과적으로 한 작업에 깊이 몰입하기 어려워하는 자각 있음

**본인이 수락한 대응 루틴 (2026-04-22)**
- **3초 캡처**: 스쳐가는 생각은 판단하지 말고 즉시 notepad에 던진 뒤 원래 작업으로 복귀 (뇌가 "이건 기억 안 해도 돼"라고 안심)
- **25분 타이머**: 끝날 때까지 절대 다른 작업으로 이동하지 않음
- **1줄 선언**: 작업 시작 전 "지금부터 X만 한다. Y/Z는 notepad에 있다"로 뇌에 신호

**협업 함의**
- 사용자가 작업에 몰입 중일 때는 현재 작업 맥락만 다뤄야 함 → `feedback_single_task_focus.md`
- 사용자는 "맥락 약화" 우려를 갖고 있지만 실제로는 깊이 파고드는 방식이 맥락을 강화한다는 설명에 납득함
