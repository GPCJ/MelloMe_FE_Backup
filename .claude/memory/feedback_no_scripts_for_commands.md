description: 스크립트를 임의로 만들지 말 것. 단, 최적화 사유가 있으면 보고 후 승인받고 생성
type: feedback
---

슬래시 커맨드 생성 요청 시 `.claude/commands/` 파일만 만들 것. 별도 스크립트 파일(scripts/*.sh 등)은 임의로 만들지 말 것.

단, 스크립트로 만들면 최적화되는 사항(예: 승인 횟수 감소, 에러 처리 강화 등)이 있다면 사용자에게 보고하고 승인받은 후 생성할 것.

**Why:** 요청하지 않은 파일을 임의로 만드는 것을 방지. 스크립트 자체가 나쁜 게 아니라 사전 승인 없이 만드는 것이 문제.

**How to apply:** 커맨드 생성 시 기본은 .md만. 스크립트가 유리한 경우 → 이유를 설명하고 승인 후 생성.
