type: project
---

현재 미인증 유저가 `/posts` 접근 시 백엔드 403 응답을 "공개 게시물이 없습니다"로 표시 중.

**Why:** 백엔드에서 공개/비공개 게시물 API를 아직 지원하지 않아, 403과 "게시물 없음"을 구분할 수 없음. 네트워크 에러 등 다른 403도 같은 메시지로 표시될 수 있음.

**How to apply:** 백엔드에서 공개 게시물 필터링 API가 추가되면, PostListPage의 403 catch 분기를 실제 공개 게시물 조회 로직으로 교체할 것.
