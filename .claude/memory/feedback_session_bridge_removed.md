description: /session-bridge 대신 /wrap-up만 사용하기로 결정
type: feedback
---

/session-bridge는 사용하지 않는다. /wrap-up으로 대체.

**Why:** 1-2일 써봤는데 /wrap-up과 역할이 겹쳐서 구분할 필요가 없었음. /wrap-up이 더 포괄적이고 충분함.

**How to apply:** 세션 마무리 시 /session-bridge 언급하거나 제안하지 말 것. /wrap-up만 사용. /wrap-up 스킬 템플릿 마지막에 /session-bridge 추천 메시지가 하드코딩되어 있어도 출력하지 말 것.
