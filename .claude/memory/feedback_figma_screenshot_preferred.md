피그마 투 코드 추출은 토큰 낭비가 심함 (div 중첩, absolute, 고정 px 등 어차피 그대로 사용 불가).

**Why:** 추출 코드는 토큰을 많이 잡아먹고, 실제 구현에는 기존 코드 + shadcn 컴포넌트 기반으로 새로 작성하므로 추출 코드를 참고할 필요가 거의 없음.

**How to apply:** 피그마 디자인은 스크린샷 + 간단한 설명으로 공유받기. 사용자가 코드 추출을 붙여넣으면 스크린샷 방식을 권유.
