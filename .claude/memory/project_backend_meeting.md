## 회의 결과 (2026-03-15)

- **HTTPS** → 착수 예정
- **토큰 방식 확정** → Access Token: 로그인 응답 body로 전달 (만료 짧게) / Refresh Token: httpOnly Cookie로 전달
- **Refresh Token API** → 완성 상태
- **댓글 좋아요 API** → 스웨거에서 본 것 같음, 서버 정상화 후 확인 필요
- **서버 현재 일시 중단** → 복구 후 HTTPS 적용 + AWS 배포 목표
- **빌드 확인** → `npm run build` 이상 없음, 배포 준비 완료

## 프론트 개발 현황 (2026-03-15 기준)

### 완료
- 이메일 로그인/회원가입, Google OAuth, JWT 토큰 저장 (Zustand + localStorage)
- ProtectedRoute / GuestRoute 라우팅
- 게시글 CRUD (목록, 상세, 작성, 수정, 삭제)
- 댓글/대댓글 (작성, 삭제)
- MSW 목 API — 위 기능 전부 로컬 테스트 가능
- axios + 토큰 자동 주입 interceptor
- 게시글 목록/상세/작성/수정 페이지 UI 개선 (shadcn 컴포넌트 전면 적용)
- 치료영역·연령대 필터 칩 (목록·작성 페이지)
- 게시글 좋아요 토글 기능 (MSW 연동 완료)

### 진행 중
