## 빌더스 리그 업로드 대상: 🤖 AI 협업 개발 역량

### 어필 포인트
- Claude Code에 외부 스킬 마켓플레이스(obra/superpowers) 플러그인을 직접 찾아서 적용
- 단순히 AI에게 "코드 짜줘"가 아니라, **개발 워크플로우 자체를 체계화**하는 도구로 활용
- 주요 활용 스킬:
  - `systematic-debugging` — 버그 발생 시 근본 원인 분석 프로세스 강제
  - `brainstorming` → `writing-plans` → `executing-plans` — 기능 구현 전 탐색→계획→실행 순서 준수
  - `verification-before-completion` — 완료 선언 전 검증 단계 자동화
  - `dispatching-parallel-agents` — 독립 작업 병렬 처리로 효율화
- SessionStart hook으로 세션 시작 시 자동 주입되도록 설정

### 면접 멘트 예시
> "AI를 코드 생성기로만 쓰는 게 아니라, 개발 프로세스(디버깅, 계획 수립, 코드 리뷰) 자체에 규율을 부여하는 도구로 활용했습니다. 외부 스킬 플러그인을 탐색해서 적용하고, 프로젝트에 맞게 커스터마이징한 경험이 있습니다."

**Why:** 면접 시 AI 활용 역량 어필용. 빌더스 리그 🤖 AI 협업 페이지에 기록 예정.

**How to apply:** 다른 기기에서 `/pull-mello` 후 `/update-builders`로 노션 업로드.
