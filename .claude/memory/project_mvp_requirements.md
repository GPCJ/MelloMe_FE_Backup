## MVP 요구 상세 (1차 초안 확정 — 2026-03-30)

### MVP 범위 (REQ-001~009)
- REQ-001: 회원가입
- REQ-002: 로그인
- REQ-003: 치료사 인증 (자격증 업로드 → 즉시 승인)
- REQ-004: 게시글 목록 (최신순/조회순 정렬, 치료 영역별 필터)
- REQ-005: 게시글 상세 (공감, 댓글 상호작용)
- REQ-006: 게시글 작성 (제목, 내용, 치료영역, 연령대)
- REQ-007: 게시글 수정/삭제
- REQ-008: 댓글 및 대댓글 작성/삭제
- REQ-009: 공감 (리액션)

## 핵심지표 (KPI)

- **WAU/MAU**: 주간/월간 활성유저
- **리텐션**: 평균 30일 잔존률
- **Uploader/User 비율**: 콘텐츠를 생산하는 치료사 비율 (커뮤니티 지속성 지표)
- **주간/월간 UCG 생성률**: 가입자 대비 주간 게시글 생성 수
- **CTR**: 콘텐츠 생성 대비 클릭 비율
- **자료 다운로드 수**
- **User 평균 30일 잔존률 (Retention)**

**Why:** MVP 성공 기준과 이후 기능 우선순위 판단에 활용
**How to apply:** 기능 개발 우선순위 논의 시, 핵심지표에 영향을 주는 기능을 우선 고려
