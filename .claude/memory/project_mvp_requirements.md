## MVP 요구 상세

### 로그인 & 인증
- 로그인 시 치료사만 들어오게 한다 (치료사 인증 필수)

### 게시판 기능
- 배너
- 검색
- 댓글
- 좋아요
- 신고
- 차단
- 알림 (비실시간)

### 글쓰기
- 필수항목: 제목, 내용 (tip tap editor)
- 선택항목: 치료영역, 파일첨부

### 기본 관리자 페이지 (프론트 화면 없어도 괜찮음)
- 치료사 인증 승인
- 신고당한 게시글 삭제
- 전체적 통계, 데이터 분석

## MVP 직후 단계

### 마이페이지
- 작성한 글
- 댓글
- 스크랩한 글

### 랜딩페이지 (커뮤니티 입장 전, 완충 — 미인증 치료사 대상)
- GNB 바
- 로그인
- 히어로섹션
- CTA 버튼 (로그인, 치료사 인증)
- 커뮤니티 설명 (효용감)
- 라스트섹션
- 푸터

## 핵심지표 (KPI)

- **WAU/MAU**: 주간/월간 활성유저
- **리텐션**: 평균 30일 잔존률
- **Uploader/User 비율**: 콘텐츠를 생산하는 치료사 비율 (커뮤니티 지속성 지표)
- **주간/월간 UCG 생성률**: 가입자 대비 주간 게시글 생성 수
- **CTR**: 콘텐츠 생성 대비 클릭 비율
- **자료 다운로드 수**
- **User 평균 30일 잔존률 (Retention)**

**Why:** MVP 성공 기준과 이후 기능 우선순위 판단에 활용
**How to apply:** 기능 개발 우선순위 논의 시, 핵심지표에 영향을 주는 기능을 우선 고려
