디자이너 협업 방식 확정 (2026-03-30)

**Why:** 프론트가 임의로 UI를 구현하는 것을 막고, 디자이너가 빠르게 필요한 UI를 확정할 수 있도록 데일리 사이클을 맞추기로 함.

**How to apply:**
- 디자이너가 당일 확정한 UI 공유 → 프론트가 구현 → 다음날 진행 상황 보고
- UI 디자인 없이는 임의 구현 금지 (타입/로직만 선구현)
- **"확정 기준 레벨" 사전 합의 필요** — 와이어프레임 수준에서 구현 시작할지, 컬러/스타일까지 확정 후 시작할지 디자이너와 명확히 맞춰둘 것. 기준 없으면 재작업 발생 가능
