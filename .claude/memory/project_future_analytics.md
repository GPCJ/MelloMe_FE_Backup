MVP 이후 사용자 데이터 수집 로직을 추가해야 함. 아직 경험 없어서 공부가 선행되어야 함.

**Why:** 서비스 성장을 위해 사용자 행동 데이터(어떤 글을 보는지, 어디서 이탈하는지 등)를 수집하고 분석하는 기능이 필요.

**How to apply:** 관련 작업 논의 시 공부가 선행되어야 함을 인지하고, 도입 방향(자체 구현 vs GA/Mixpanel 등 외부 툴) 같이 검토할 것. 개발 중 데이터 수집에 유리한 구조(컴포넌트 분리, 이벤트 훅 등)를 의식적으로 고려할 것.
