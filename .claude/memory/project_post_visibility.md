**결정 (2026-03-24, 업데이트):** 비로그인 또는 치료사 인증 미완료 유저는 게시글 열람 불가 → 랜딩페이지로 리다이렉트.

**Why:** 멜로미는 검증된 치료사들만의 커뮤니티가 핵심. 인증이 서비스 신뢰도의 근간이므로 미인증 유저의 커뮤니티 접근 차단.

**How to apply:**
- ProtectedRoute에서 로그인 여부 + 인증 여부(isVerified 등) 둘 다 체크
- 비로그인 또는 미인증 → `/` (랜딩페이지) 리다이렉트
- 랜딩페이지: 서비스 소개 + 회원가입/로그인/치료사 인증 유도 CTA 포함
- LandingPage 신규 생성 필요 (초안 피그마 기준)
- 기존 정책(로그인만 체크) → 인증 여부까지 체크하도록 ProtectedRoute 수정 필요
