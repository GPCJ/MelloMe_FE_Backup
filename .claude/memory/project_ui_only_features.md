## 로그인 페이지 (`/login`)

- **로그인 상태 유지** 체크박스 — UI만 존재, 기능 없음
  - 실제 동작하려면 백엔드에서 Set-Cookie의 Max-Age를 체크 여부에 따라 다르게 설정해야 함
  - 프론트에서 체크 여부를 요청 body에 담아 전송하는 방식

- **비밀번호 찾기** 링크 — UI만 존재, 기능 없음

- **이용약관 / 개인정보처리방침** 링크 — UI만 존재, 연결된 페이지 없음

## 커뮤니티 목록 페이지 (`/posts`)

- **검색바** — UI만 존재, 기능 없음
  - 실제 동작하려면 백엔드에 검색 API 필요

- **배너 통계** (이번 주 게시글 N개, 누적 공감 N개) — 하드코딩된 숫자, API 연결 안 됨

## 글쓰기 / 수정 페이지 (`/posts/new`, `/posts/:id/edit`)

- **파일 첨부 영역** — UI만 존재, 실제 업로드 기능 없음
  - 실제 동작하려면 파일 업로드 API 및 백엔드 스토리지 연동 필요

## 네비게이션 바

- **공지사항 드롭다운** — mock 데이터로 고정 표시, 실제 API 연결 안 됨
  - 백엔드 알림 기능 완성 시 연동 예정
  - "전체 공지사항 보기" 버튼 클릭 시 아무 동작 없음

**How to apply:** 위 기능들 관련 이야기가 나오면 "현재 UI만 있고 동작은 안 합니다"라고 먼저 알릴 것.
