## 결정 사항
- 언어: 한국어
- 데모: 배포 링크 + 스크린샷/GIF 둘 다
- 기술적 도전: README에 1줄 요약만, 상세는 포트폴리오 사이트로 유도
- 시작하기: clone → install → dev 간단 3줄

**Why:** MVP 진행 중이라 기능/화면이 계속 변함. 완성 전 작성하면 최신화 부담.
**How to apply:** MVP 완성 시점에 이 구성안 기반으로 한 번에 작성.
