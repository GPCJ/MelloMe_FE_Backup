---
name: 취업용 경험 Notion 정리 제안
description: 취업에 활용할 만한 개발 경험이 생기면 Notion 정리를 제안할 것
type: feedback
---

취업할 때 써먹을 수 있는 경험(트러블슈팅, 기술 결정, 성능 개선 등)이 발생하면 Notion에 정리하는 게 어떠냐고 물어볼 것.

**Why:** 사용자가 이 프로젝트 경험을 취업에 활용할 계획. 개발하면서 지나치기 쉬운 경험들을 포트폴리오/면접 답변으로 남겨두길 원함.

**How to apply:** 아래 상황에서 "이 경험 Notion에 정리해두는 게 어떠세요?"라고 제안할 것.
- 버그를 원인까지 파악해서 해결했을 때
- 기술 선택 이유를 설명할 수 있는 결정을 내렸을 때 (예: shadcn 도입, MSW 도입 등)
- 아키텍처 설계 고민을 했을 때 (예: ProtectedRoute 구조, 전역 상태 설계)
- 배포/인프라 관련 문제를 해결했을 때 (예: Vercel SPA 404 문제)
