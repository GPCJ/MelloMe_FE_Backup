이번 대화 내용과 메모리를 바탕으로 노션 진행 상황 페이지를 업데이트해줘.

노션 페이지 URL: https://www.notion.so/322c8200749b81a2a188dd0082b6b15a

## 작업 순서

1. 노션 페이지를 fetch해서 현재 내용 확인
2. 메모리에서 현재 진행 상황 파악
3. 이번 대화에서 작업한 내용 정리
4. 아래 기준에 따라 업데이트할 내용 초안을 채팅창에 먼저 출력
   - "현재 진행 중" 섹션 변경사항
   - "주간 기록"에 추가할 항목
5. 사용자 확인 후 승인이 오면 그때 노션 페이지 업데이트

## 업데이트 기준

### 현재 진행 중 섹션
- 지금 이 시점에 진행 중이거나 대기 중인 작업만 남겨둘 것
- 완료된 항목은 제거

### 주간 기록 섹션
- 오늘 날짜 기준으로 해당 주차 항목에 이번 대화에서 한 작업 추가
- 해당 주차가 없으면 새로 만들 것 (형식: `## YYYY-MM-DD ~ MM-DD`)
- 이미 기록된 내용과 중복되지 않게 할 것

## 작성 원칙 (중요)

팀원 중 개발자가 아닌 사람도 읽는다. 아래 원칙을 반드시 지킬 것:

- 기술 용어 대신 기능/목적 중심으로 서술
- "무엇을 했고, 왜 필요한지"가 드러나도록
- 한 줄로 핵심만

**나쁜 예시**:
- `Google OAuth 백엔드 주도 전환, OAuthCallbackPage 미구현`
- `withCredentials CORS preflight 이슈 확인`

**좋은 예시**:
- `Google 로그인 방식을 더 안전하게 개선 (백엔드 팀과 마지막 연결 방식 확정 후 완료 예정)`
- `Google 로그인 시 인증 오류 원인 파악 및 백엔드 수정 요청`
