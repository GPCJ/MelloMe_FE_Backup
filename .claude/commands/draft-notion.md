이번 대화 내용을 노션에 올릴 초안으로 정리해서 메모리에 저장해줘.

## 용도

현재 기기에서 노션 접근 권한이 없을 때, 초안을 메모리에 저장해두고 다른 기기에서 `/pull-mello` 후 노션에 직접 올리기 위한 커맨드.

## 작업 순서

1. 이번 대화에서 노션에 기록할 만한 내용 정리 (작업 내용, 결정사항, 점검 결과 등)
2. 메모리에서 대상 노션 페이지 URL 확인 (reference_notion_*.md 파일들 참조)
3. 아래 형식으로 초안을 채팅창에 먼저 출력
4. 사용자 확인 후 승인이 오면 `.claude/memory/notion_draft.md`에 저장 (기존 내용 덮어쓰기)
5. 저장 완료 후 안내: "다른 기기에서 `/pull-mello` 후 `notion_draft.md` 확인하세요"

## 초안 형식

```
# 노션 업데이트 초안 (YYYY-MM-DD)

**대상 페이지:** 페이지 이름
**URL:** 노션 페이지 URL

---

(대상 페이지에 맞는 형태로 자유롭게 작성)
```

## 대상 페이지 판단 기준

- 작업 진행 상황 → 멜로미 프론트엔드 개발 진행 상황 (reference_notion_progress.md)
- 오늘 배운 것 → TIL 페이지 (reference_notion_til.md) — `/draft-til` 사용 권장
- 트러블슈팅 → 트러블슈팅 페이지 (reference_notion_troubleshooting.md)
- 사용자가 직접 대상 페이지를 지정하면 그대로 따를 것

## 작성 원칙

- 팀원 중 개발자가 아닌 사람도 읽는다는 전제
- 기술 용어 대신 기능/목적 중심으로 서술
- "무엇을 했고, 왜 필요한지"가 드러나도록
- 한 줄로 핵심만
