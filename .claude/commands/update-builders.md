## 초안 작성 우선순위

**Step 1**: `.claude/memory/til_draft.md` 파일을 먼저 읽어라.

- 파일이 존재하고 오늘 날짜(YYYY-MM-DD)의 내용이 있으면 → 해당 내용을 기반으로 초안 작성
- 파일이 없거나 오늘 날짜 내용이 없으면 → 이번 대화 내용과 메모리를 바탕으로 초안 작성

**Step 2**: 노션 페이지를 fetch해서 오늘 날짜와 동일한 제목의 서브페이지가 이미 존재하는지 확인해라.

- 이미 존재하면 → "오늘 날짜 항목이 이미 노션에 존재합니다. 이번 세션 내용으로 새로 작성할까요?" 라고 물어보고 승인 시에만 진행
- 존재하지 않으면 → 초안 보여주고 승인 후 업로드

## 기록 방식
- 기존 TIL 양식에 따라 오늘 날짜 항목을 서브페이지로 추가
- 아래 항목을 반드시 포함:

```
## YYYY-MM-DD — 제목

**분류**: React / TypeScript / 인프라 / 보안 / AI활용 / 기타

### 오늘 한 것
- 오늘 작업 내용 bullet point 요약

### 배운 것 / 인사이트
- 새로 알게 된 것, 깨달은 것

### 포트폴리오 어필 포인트 (있을 경우만)
- 이력서/면접에 쓸만한 내용
```

빌더스 리그 트러블슈팅(🛠) 섹션이 til_draft.md에 있으면 함께 포함할 것.

## 주의사항
- 단순 반복 작업(push, 파일 정리 등)은 생략하고 의미 있는 작업만 기록
- 업로드 전 채팅창에 초안 보여주고 승인 후 업로드
