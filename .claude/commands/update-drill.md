`docs/coding-drill-log.md`를 읽고 이번 주 코딩 드릴 기록을 노션에 업로드해줘.

노션 코딩 드릴 페이지 URL: https://www.notion.so/323c8200749b80c2bbe6caf194055593
(TIL 페이지 하위에 "코딩 드릴" 서브페이지로 생성)

## 작업 순서

1. `docs/coding-drill-log.md` 파일을 읽어서 이번 주 기록 확인
2. 노션 TIL 페이지를 fetch해서 "코딩 드릴" 서브페이지가 있는지 확인
3. 없으면 서브페이지 생성, 있으면 기존 페이지에 추가
4. 아래 양식으로 초안을 채팅창에 먼저 출력
5. 사용자 확인 후 승인이 오면 그때 노션 업데이트

## 주간 리포트 양식

```
## W{주차} (MM-DD ~ MM-DD)

### 이번 주 드릴 기록
| Day | 파일 | 수준 | 막힌 곳 |
|-----|------|------|---------|
| ... | ...  | ...  | ...     |

### 주간 분석
- 평균 수준: ★★☆
- 강한 영역: (잘한 부분)
- 보완 필요: (반복적으로 막힌 부분)

### 다음 주 추천
- (난이도 조정 제안 + 추천 파일)
```

## 작성 원칙

- 기록이 없으면 "이번 주 드릴 기록이 없습니다" 알려주기
- 주간 분석은 패턴 위주로 간결하게
- 다음 주 추천은 coding-drill-guide.md의 난이도별 파일 풀에서 선택
